About Us

SERA — Smart Emotion Recognition Assistant is a Semester-3 capstone project developed at "Ratan Tata Maharashtra State Skills University". Our goal is to bridge technical learning with practical impact: we build a lightweight, student-friendly speech-emotion detection tool that demonstrates ML pipelines, real-time audio capture and user-focused UI in a single polished project.

This project showcases end-to-end work: dataset handling (RAVDESS / TESS), feature extraction, model training, and a responsive web interface that accepts live audio and uploaded files.

We build, we test, we learn — and we make every demo count.
